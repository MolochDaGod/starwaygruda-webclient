Troubleshooting:
========================================

Server Not Loading?
1. Check PowerShell window is still open
2. Look for any error messages
3. Try refreshing browser (Ctrl+F5)

Port Changed?
- Server auto-tries ports 8080, 8081, 8082, 8083
- Check PowerShell window for actual URL

Need to Restart?
1. Close PowerShell window
2. Run: npm run dev
   Or: START_ALL.bat
