## ğŸ› Troubleshooting

**Nothing loads?**
- Check browser console (F12)
- Make sure you're on game.html, not index.html
- Refresh the page

**Minimap not showing?**
- Press **M** key to toggle
- Check console for errors
- Make sure planet has loaded

**Hotkeys not working?**
- Click on the game window first
- Don't type in input fields
- Press **H** to verify system is active

**Fast travel menu empty?**
- Make sure a planet is loaded
- Check that admin mode is enabled
- Press **T** again to refresh

**Performance issues?**
- Close other browser tabs
- Lower camera distance (zoom in)
- Use a less populated planet (Lok, Rori)
